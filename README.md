## 📌 Project Overview

This project showcases an **interactive Power BI dashboard** built to analyze sales data for **Blinkit (Quick Commerce Platform)**.
It helps uncover **business insights, trends, and performance metrics** to support smarter decision-making.

---

## 🎯 Key Objectives

✔ Analyze overall sales performance
✔ Identify top-performing products & categories
✔ Track revenue trends over time
✔ Understand customer purchasing behavior
✔ Deliver actionable insights

---

## 🛠️ Tech Stack

* 📊 Power BI (Data Visualization)
* 🔄 Power Query (Data Cleaning & Transformation)
* 🧮 DAX (Calculations & Measures)
* 📁 Excel / CSV (Dataset)
